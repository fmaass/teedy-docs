# Teedy Roadmap

The in-repo roadmap has been retired to keep a single source of truth and avoid
three-way drift between this file, git history, and the maintainer's planning wiki.

- **Planned work & feature dispositions** are tracked in the maintainer's internal
  ARCHIMEDES/BookStack wiki (Fork Remediation Plan). This is the authoritative
  source and is not part of the public fork.
- **Shipped changes** — see the public changelog on the
  [GitHub releases page](https://github.com/fmaass/teedy-docs/releases). Each
  release tag documents what changed.

For architecture and feature docs, see `README.md` and `CLAUDE.md`.
